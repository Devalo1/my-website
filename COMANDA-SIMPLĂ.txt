PRIMA COMANDĂ PENTRU NAVIGARE LA DIRECTORUL PROIECTULUI

Când Command Prompt se deschide în c:\Users\support, folosește această comandă pentru a naviga la folderul proiectului:

cd /d c:\Proiect\my-website

Explicație:
- "cd" înseamnă "change directory" (schimbă directorul)
- "/d" este un parametru care îți permite să schimbi și discul, nu doar directorul
- "c:\Proiect\my-website" este calea completă către directorul proiectului tău

După ce ai executat această comandă, vei fi în directorul proiectului și poți rula orice script dorești.

Exemplu complet:
1. Deschide Command Prompt (cmd)
2. Vei vedea: C:\Users\support>
3. Tastează: cd /d c:\Proiect\my-website
4. Apasă Enter
5. Vei vedea: C:\Proiect\my-website>
6. Acum poți rula orice script, de exemplu: fix-git-head.bat

În alternativă, poți folosi metoda mai simplă:
1. Deschide Windows Explorer
2. Navighează la folderul c:\Proiect\my-website
3. Fă dublu-click pe fișierul deschide-terminal-aici.bat
4. Se va deschide automat un Command Prompt în directorul corect
